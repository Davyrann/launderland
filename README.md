# LaunderLand
Aplikasi manajemen laundry minimalis yang dirancang khusus untuk digunakan oleh pemilik (owner) laundry. Aplikasi ini mempermudah 
pencatatan transaksi masuk hingga pelacakan status cucian yang sudah selesai tanpa ribet.

## Fitur Utama
- Buat Pesanan Baru
- Selesaikan Pesanan
